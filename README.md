# Software Engineering Assignments – ObreshkovLibrary Project

## Описание

Това repository съдържа документацията към дипломния проект ObreshkovLibrary, разработена по заданията по предмет „Софтуерно инженерство“ в XII клас, специалност „Приложен програмист“.

Документите проследяват основните етапи от разработката на софтуерен проект:
- анализ и планиране;
- проектиране и архитектура;
- реализация;
- тестване и документация.

## Цел

Целта на това repository е да представи процеса на разработка на дипломния проект чрез структурирана техническа документация съгласно изискванията на учебната програма.

## Структура на repository-то

В папка `docs/` се съхраняват документите към заданията:

- `01-project-analysis.md` – анализ и планиране на проекта
- `02-design-and-architecture.md` – архитектура и модел на системата
- `03-implementation-report.md` – описание на реализацията (предстои)
- `04-testing-and-documentation.md` – тестване и финална документация (предстои)

Допълнително е приложена ER диаграма на базата данни:

- `database-diagram.png`

## Свързан проект

Документацията е създадена за дипломния проект:

**ObreshkovLibrary** – уеб базирана система за управление на училищна библиотека, разработена с ASP.NET Core MVC и SQL Server.

## Използвани технологии в проекта

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript
- Bootstrap
- ASP.NET Core Identity

## Начин на използване

Документите описват архитектурата, структурата и развитието на дипломния проект по етапи съгласно заданията по софтуерно инженерство.
